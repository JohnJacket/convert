System requirements:
--------------------

* GTK 3 and GTK 3 dev.
* gstreamer 1.0 and above (dev package for Eclipse).
* gcc 4.7 and above.

// Linux installation.
======================

To create an executable run makefile:
   > make all.
In case no errors you get convert.

// Using Eclipse IDE.
=====================

To open convert project in Eclipse IDE you can Import convert 
into workspace:
    > File -> Import... -> Existing projects into workspace.
Set the root directory to where your .project and .cproject 
files are located. Your project should show up in the list.

Then compile and run convert.
